1. Introduction

The GTECH is an educational platform designed to provide interactive learning materials related to multimedia and digital design. The website allows users to explore lessons and view infographics easily through a simple navigation system.

2. Website Navigation

The navigation bar located at the top of the website allows users to easily move between different pages.

  • Home – Returns the user to the main page.
  • About Us – Shows information about the website.
  • Manual – Displays the user manual page.
  • Book – Opens a dropdown menu containing lessons and infographics.

3. Learning Cards

The homepage contains interactive cards that represent different learning topics available on the website.

  • Multimedia
  • Audio
  • Color Theory
  • Adobe Photoshop
  • Animation

When a user clicks a card, the system will redirect them to the corresponding lesson page.

4. Lessons and Infographics

Users can access lessons and infographic materials through the Book dropdown menu or by selecting a learning card. These materials contain step-by-step explanations and visual learning guides.

5. Contact Form

The website provides a contact form where users can send questions or inquiries to the administrator.

  • Enter your name
  • Enter your phone number
  • Write your message
  • Click the Send button

The system checks if all fields are filled before sending the message. If any field is empty, the form will show a warning message.

6. Responsive Design

The website is designed to work on different screen sizes including desktop computers, tablets, and mobile phones. The layout automatically adjusts to provide a better viewing experience.

7. Conclusion

The GTECH is a learning website provides a simple and organized platform where users can access multimedia learning materials, interactive visuals, and educational resources.
